# Software Engineering

## 1. 概要 (Description)

本システムは、大学空手部における**2拠点（船橋・駿河台キャンパス）に分散した防具（面、胴、拳サポ等）の在庫・所在をリアルタイムに一元管理するシステム**です。

### 🚨 解決したい課題
* **拠点の分散:** 船橋と駿河台の2箇所に防具があるため、正確な在庫数の把握が困難。
* **遠征準備の負担:** 試合や遠征の前、練習終了後の限られた時間（約1.5時間）で手作業によるカウントと準備を行っており、部員の大きな負担となっている。

### ✨ 本システムによる解決（メリット）
各防具に貼付した**QRコードをスマホでスキャンし、移動先（船橋 / 駿河台 / 遠征）を選択するだけ**で、所在を瞬時にアップデートできます。手作業のカウントを無くし、遠征準備にかかる時間を劇的に削減します。

### 🛠️ 主な機能
* **QRコードスキャン:** スマホカメラで防具のQRを読み取り、個体を識別（手入力バックアップあり）。
* **リアルタイム在庫表示:** 「船橋」「駿河台」「遠征（持ち出し中）」の3拠点ごとの在庫数を一目で確認。
* **超シンプルUI:** 練習後の疲れた状態でも迷わず、最小限のタップ数でステータス更新が完結。
